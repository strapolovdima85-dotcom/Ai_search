# Программный комплекс «Интеллектуальный поиск номенклатуры в 1С:ERP»

Данный проект разработан в рамках дипломной работы на курсе **«Архитектор ИИ»**. Система внедряет гибридный семантический поиск по смыслу, синонимам и техническому описанию в стандартный интерфейс подбора номенклатуры 1С:ERP Управление предприятием.

## 🚀 Быстрый запуск комплекса

Вся инфраструктура (ИИ-сервис на FastAPI и векторная база данных Qdrant) полностью контейнеризирована и запускается одной командой.

### Шаг 1. Подготовка окружения
Убедитесь, что на вашем сервере или рабочем компьютере установлен и запущен **Docker Desktop** (или Docker Daemon с Docker Compose).

### Шаг 2. Развертывание контейнеров
Откройте терминал (PowerShell или Command Prompt) в корневой папке проекта (`Ai_search/`) и выполните команду:

```bash
docker compose up -d --build
```

**Что произойдет при первом запуске:**
1. Развернется контейнер векторной базы данных `Qdrant` на портах `6333` и `6334`.
2. Соберется образ ИИ-микросервиса на базе легковесного `python:3.11-slim`.
3. Внутри контейнера автоматически скачается русскоязычная языковая модель `RuBERT` с репозитория Hugging Face и сохранится в локальный кэш на жестком диске.
4. Создастся поисковая коллекция `nomenclature` с косинусной метрикой близости векторов.

### Шаг 3. Проверка работоспособности
Откройте браузер и перейдите по адресу интерактивной документации Swagger UI:
👉 **`http://localhost:8000/docs`**

Вы сможете протестировать доступность ИИ-сервиса через эндпоинт `/health`.

## 💻 Порядок интеграции и работы в 1С:ERP

Для демонстрации работы системы на защите диплома выполните следующие шаги внутри 1С:

### 1. Подготовка внешней обработки
1. Откройте **1С:Предприятие** в режиме **Конфигуратор**.
2. Создайте новую Внешнюю обработку, добавьте форму.
3. В Конструкторе формы создайте реквизиты (`СтрокаПоиска`, `СписокНоменклатуры`, `АдресСервера`, `ПортСервера`) согласно инструкции в файле документации проекта.
4. Запустите обработку в режиме «1С:Предприятие» `1c_components/Ai_search.epf`.


### 2. Шаг Первичной Синхронизации (Offline)
1. В интерфейсе открытой обработки 1С убедитесь, что в полях подключения указаны `127.0.0.1` и порт `8000`.
2. Нажмите кнопку **«Выгрузить всю номенклатуру»**.
3. Система выполнит оптимизированный запрос, соберет данные полей `Наименование`, `Артикул`, `Описание` и `НаименованиеПолное`, склеит их в единый контекст и порциями (батчами по 100 штук) отправит в Docker-контейнер.
4. ИИ-сервис преобразует тексты в векторы размерностью 768 и сохранит их в Qdrant. База проиндексирована.

### 3. Тестирование умного поиска (Online)
Введите в поле **Строка поиска** любой сложный, неточный запрос или профессиональный сленг вашей отрасли, например:
* *«саморез гд черный 19»* (изменение порядка слов, сокращения).
* *«cfvjhtp 3.5»* (ошибка раскладки клавиатуры).
* *«нагнетатель жидкости»* (поиск по смыслу и синонимам вместо точного слова "насос").

Интерфейс 1С мгновенно (менее чем за 250 мс) наложит динамический отбор на справочник номенклатуры, выведя ТОП релевантных позиций и их аналогов, найденных искусственным интеллектом.
